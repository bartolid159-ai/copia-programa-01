# Tarea 01: Inicialización del Proyecto y Configuración Base

## Objetivo
Configurar el entorno de desarrollo con Vite + React y preparar la estructura de archivos base siguiendo la estética Premium.

## Archivos Afectados
- `package.json`
- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/index.css` (Diseño de tokens y sistema visual)

## Criterio de Éxito
- El sistema corre con `npm run dev`.
- Se visualiza una interfaz básica con el esquema de colores (Dark Mode) y tipografía definidos en `visual_specs.md`.
