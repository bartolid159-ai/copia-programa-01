# Tarea 03: Módulo de Gestión de Pacientes

## Objetivo
Desarrollar la interfaz CRUD de pacientes con búsqueda predictiva en tiempo real por Cédula o Nombre.

## Archivos Afectados
- `src/components/Patients/PatientList.jsx`
- `src/components/Patients/PatientForm.jsx` (Demás campos: Sexo, DOB, Dirección)
- `src/logic/patientService.js`

## Criterio de Éxito
- Se pueden registrar pacientes con los 7 campos requeridos.
- La barra de búsqueda muestra resultados dinámicos al escribir.
- No se permiten cédulas duplicadas.
