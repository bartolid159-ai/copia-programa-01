# Módulo: Contabilidad y Reportes Avanzados

## Segmentación de Datos (Dashboard y Reportes)
- **Por Servicio:** Ranking de rentabilidad.
- **Por Médico:** Sumatoria de comisiones y facturación bruta.
- **Por Tiempo:** Filtros por Día, Semana, Mes y Año.
- **Por Categoría:** Diferenciación entre Servicios Médicos e Insumos vendidos por separado.

## KPIs en Dashboard
1. **Ganancia Neta:** Ingresos Brutos - Comisiones - Costos de Insumos.
2. **Top 5 Servicios:** Gráfica de Pareto.
3. **Stock Crítico:** Widget de alerta para insumos bajo el `stock_minimo`.
4. **Flujo de Caja:** Gráfica de Ingresos vs. Egresos mensuales.

## Reglas de Inmutabilidad
- Las facturas ya cerradas no se editan. Los ajustes se hacen vía Notas de Crédito/Débito.
