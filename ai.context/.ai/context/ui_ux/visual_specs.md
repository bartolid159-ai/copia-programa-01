# Especificaciones Visuales (UI/UX)

## Estética General
- **Concepto:** Clínica Digital Moderna / Premium.
- **Modo:** Oscuro (Dark Mode) por defecto (según PRD "Moderna").
- **Estilo:** Glassmorphism sutil, bordes redondeados (8px a 12px), sombras suaves.

## Tipografía
- **Fuente Principal:** Inter o Roboto (Google Fonts).
- **Legibilidad:** Énfasis en alto contraste para la carga rápida de datos.

## Paleta de Colores
- **Primario:** Teal / Azul Clínico Profundo (#0D9488 o similar).
- **Fondo:** Dark Grey / Deep Blue para el modo oscuro (#0F172A).
- **Acentos:** Verde Esmeralda para "Pagado" e Inventario Positivo, Rojo vibrante para "Stock Crítico" y Egresos.

## Componentes de Interfaz
- **Buscadores:** Input con icono de lupa, dropdown de resultados con filtrado en tiempo real.
- **Formularios:** Etiquetas claras, validación visual instantánea.
- **Gráficas:** Uso de librerías como Chart.js o Recharts con colores coherentes a la marca.

## UX Principles
- **Quick-Input:** El tabulador debe seguir el flujo lógico de la factura.
- **Feedback:** Notificaciones (toasts) para acciones exitosas (Factura Guardada, Stock Actualizado).
