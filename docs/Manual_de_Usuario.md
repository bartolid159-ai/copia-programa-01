# Manual de Usuario - Sistema de Gestión Médica

Bienvenido al **Sistema de Gestión Médica**. Este programa ha sido diseñado para funcionar de manera 100% local, garantizando la privacidad de sus datos.

## Funciones Principales

### 1. Panel de Contabilidad
- Visualice el resumen de ingresos y gastos.
- Gráficos interactivos de rendimiento mensual.
- Alertas de stock bajo para insumos médicos.

### 2. Gestión de Pacientes
- Registro completo de historias clínicas.
- Búsqueda rápida por nombre o identificación.
- Historial de facturas asociadas a cada paciente.

### 3. Facturación y Cobros
- Generación de facturas profesionales.
- Soporte para múltiples divisas (USD/VES).
- Sistema de liquidación para médicos (comisiones).

### 4. Seguridad y Backups
- El sistema realiza copias de seguridad automáticas localmente.
- Acceso restringido mediante usuario y contraseña.
